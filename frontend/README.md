# GlobalMarketHub Frontend

React-based customer portal for GlobalMarketHub multi-marketplace platform.

## Features

- Product marketplace browsing
- Professional services discovery
- Import/Export B2B platform
- User authentication
- Responsive design
- Real-time search

## Installation

```bash
npm install
```

## Running

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in browser.

## Build

```bash
npm run build
```
